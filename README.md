# Heart Disease Prediction App ❤️

A Machine Learning-powered web application that predicts the likelihood of heart disease based on medical attributes. Built using Python, Scikit-learn, and Streamlit for an interactive and user-friendly experience.

---

## 🚀 Live Demo

🌐 https://heartdiseasepredictionapp-rfygncdygxxk25yjbjpoyb.streamlit.app/

---

## 📌 Features

- Predicts heart disease risk instantly
- Clean and interactive Streamlit UI
- Trained Machine Learning classification model
- User-friendly input form
- Fast and lightweight deployment

## ⚙️ Installation & Setup

### Clone the Repository

git clone https://github.com/AyushChaki/Heart_DiseasePrediction_App.git

### Navigate to Project Folder

cd Heart_DiseasePrediction_App

### Install Dependencies

pip install -r requirements.txt

### Run the Application

streamlit run app.py

---

## 📈 Model Information

- Algorithm Used: K-Nearest Neighbors (KNN)
- Problem Type: Classification
- Target Variable: Heart Disease Presence
